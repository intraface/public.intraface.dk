Intraface PEAR channel

Registering the channel: 
pear channel-discover public.intraface.dk
Listing available packages: 
pear remote-list -c intrafacepublic
Installing a package: 
pear install intrafacepublic/package_name
Installing a specific version/stability: 
pear install intrafacepublic/package_name-1.0.0
pear install intrafacepublic/package_name-beta
Receiving updates via a feed: 
http://public.intraface.dk/feed.xml

The public.intraface.dk PEAR Channel Server is proudly powered by Pirum 1.1.5
